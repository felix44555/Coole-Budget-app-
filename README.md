# Budget.

Mobiler Haushaltsrechner — verrechnet Einnahmen und Ausgaben und prognostiziert, wie viel am Ende des Monats ubrig bleibt.

## Features

- Einnahmen und Ausgaben erfassen
- Wiederkehrende Posten fur Monatsprognose
- Ausgabenanalyse nach Kategorie
- Kennzahlen: Tagesschnitt, Sparquote, Fixkostenanteil
- Daten werden lokal im Browser gespeichert (localStorage)
- Handy-optimiert

## Setup

```bash
npm install
npm run dev
```

## Deploy (GitHub Pages)

Der GitHub Actions Workflow deployt automatisch bei jedem Push auf `main`.

Einmalig in den Repo-Einstellungen aktivieren:
Settings > Pages > Source: GitHub Actions
